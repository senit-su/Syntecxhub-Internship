
# TCP Port Scanner (Python)

## Description
A Python-based TCP port scanner that identifies open, closed, and filtered ports on a target host using socket programming and multithreading.
Built to learn basic networking and cybersecurity concepts.

## Features
* Scan a single host
* Custom port range
* Multithreaded scanning
* Console output and log file
* Handles timeouts and errors

## Tech Stack
* Python 3
* Socket Programming
* Multithreading
* Logging

## Usage
Change directory to where your file is
```
cd "C:\Users\HP\Documents\Syntecxhub Internship"
```
Run the scanner
```
python tcp_port_scanner.py
```
Example input:
```
Host: 127.0.0.1
Start Port: 20
End Port: 100
```
## Sample Output
```
[OPEN] 22
[CLOSED] 23
[TIMEOUT] 80
```
